# LogiTrack 使い方マニュアル

本番URL: https://unkou-final.web.app

画面が古いまま変わらない時は、`Ctrl + Shift + R`（強制再読み込み）を押してください。

---

## 1. 画面の全体構成
左メニューで画面を切り替えます。

| メニュー | 内容 |
|---|---|
| ダッシュボード | KPI・監視マップ・運搬ロット一覧・アラート |
| 船舶位置 | 船の稼働状況 |
| ダンプ位置 | ダンプのカンバン（待機 / 輸送中） |
| 運搬履歴 | 完了したロットの一覧 |
| 港入港スケジュール | 相馬港バース管理（PDF読み取り・カレンダー編集） |
| ライブカメラ | 配信映像の表示 |

---

## 2. ロットを登録する（船・ダンプ共通）
1. ダッシュボード右上の「**＋ 新規ロット追加**」を押す
2. 「種別」で **船舶** か **ダンプ** を選ぶ
3. 会社名・船名/車両番号・現在地・最終埋立現場・残土情報（総トン数/重量トン数/数量m³）などを入力
   - 数値を入れると、ダッシュボードの「総積込量」「最終埋立完了」KPIに反映されます
4. 「登録する」を押す

### ダンプ特有の項目
- **配車予定日** … この日の朝7時に自動で「輸送中」へ進みます（後述）
- **ドライバー名 / 積載(m³) / 車両区分** など

## 3. ダンプの入場登録（待機）
1. 「**＋ 入場登録**」を押す
2. 車両ナンバー・積載容量・所属会社・ドライバーを入力 → 「入場登録する」
3. ダッシュボードの「入場登録済み（配車待ち）」と、「ダンプ位置」の **待機** 列に並びます

---

## 4. ダンプの自動ステータス遷移（重要）
ダンプは**時刻に応じて自動で進みます**。

1. 登録した翌日（または入力した **配車予定日**）の **朝7:00** → 「**輸送中**」へ
2. その日の **夜19:00** → 「**運搬履歴**」へ

ポイント:
- 配車予定日を入力すればその日が基準。未入力なら「登録の翌日」が基準になります。
- アプリを開いていない時間帯は処理されませんが、**次に誰かが開いた時点で正しい段階に追いつき**ます（例：夜に閉じていても翌朝開けば履歴に移動済み）。
- 「積込中」「荷卸し中」など手動で付けた途中ステータスは自動で上書きしません。

---

## 5. ダッシュボードの見方
- **KPIカード** … 実データから自動計算
  - 総積込量 = 進行中ロット＋完了履歴の合計トン数
  - 船舶/ダンプ稼働数 = 稼働中 ÷ 登録台数
  - 最終埋立完了 = 埋立地に届いた合計トン数（進捗バー付き）
- **リアルタイム監視マップ** … 積込港→荷下ろし→最終埋立地の流れを図示
- **運搬ロット（船 / ダンプ）** … 一覧。行の「編集」「配車情報」から修正
- **アラート** … 自動生成される通知
- **CSV出力 / 印刷** … 右上のボタン

---

## 6. 港入港スケジュール（相馬港バース管理）
左メニュー「港入港スケジュール」で開きます。船の入出港予定をバース×日付で管理します。

### 6-1. PDFをAIで自動読み取り
1. 左の「**資料・自動読取**」を開く
2. 「**ファイルを選択**」で入出港予定表のPDFを選ぶ
3. 右上の解析エンジンが「**ChatGPT(OpenAI)**」になっているのを確認
4. 「**✨ AIで上書き反映**」を押す → 「AI解析中…」が出る（10〜30秒）
5. 解析が終わると**その予定の月のカレンダー**に自動で切り替わります

> 表が非常に密なため、AIは100%完璧には読めません。読めなかった分は次の手編集で直します。

### 6-2. 表示の切替・移動
- 「**月間 / 日次**」で表示を切替
- 上部中央の**日付欄をクリック**するとカレンダーから日付を直接選べます
- ◀ ▶ ボタンは、月間表示では「前後の月」、日次表示では「前後の日」に移動

### 6-3. ドラッグ＆ドロップで移動（Googleカレンダー風）
- 月間表示で、**船名のタグをつかんで別の日へドロップ**すると日付が移動します
- ドロップ先が緑にハイライトされます

### 6-4. クリックで手編集 / 追加 / 削除
- 船名タグを**クリック**すると右に**編集フォーム**が開きます
  - 日付・バース・船名・入出港時刻・貨物・状況を修正 →「**保存して反映**」で正しい位置へ移動
  - 「**削除**」で間違った予定を消去
- 右パネルの「**＋ 新規追加**」で、抜けている船を手入力で追加

### 6-5. 変更履歴
- 右上の「**🕘 変更履歴**」で、いつ・どの船を・どこから→どこへ動かしたかの一覧を確認できます
- ドラッグ移動・編集での日付/バース変更が自動で記録されます

---

## 7. ライブカメラ
1. 「ライブカメラ」を開く
2. 配信URL（**HLS（.m3u8）** または **YouTube等の埋め込みURL**）を入力
3. 「表示」を押す
> RTSPのURLはブラウザで直接再生できません。HLS/WebRTC等への変換が必要です。

---

## 8. 困ったとき
- 画面が更新されない → `Ctrl + Shift + R`
- PDFを読み取っても出てこない → カレンダーが**別の月**になっていないか確認（その予定の月へ移動）
- OCRでエラーが出る → エラー文を控えて管理者へ（Functionsのログから原因を確認できます）
- データが保存されない → ネットワーク / Firestore の状態を確認
